# E-commerce Demand & Pricing Optimizer (End-to-End)

## Business Problem

Este proyecto resuelve el desafío de predecir la demanda de productos para un e-commerce real (Olist Dataset), permitiendo la optimización de inventarios y estrategias de precios dinámicos.

## 🛠 Tech Stack

- **Data:** PostgreSQL (Relational DB), SQL.
- **Languages/Tools:** Python (Pandas, Scikit-learn, XGBoost).
- **MLOps:** MLflow (Experiment Tracking), Docker (Containerization).
- **API/Deployment:** FastAPI, Google Cloud Platform (Cloud Run).
- **UI/Viz:** Streamlit / Plotly.
